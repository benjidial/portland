  Info passed from bootloader

0x0100 - 0x0546 (1095B):
  Kernel variables

0x0547 - 0x???? (?B):
  Kernel code

0x???? - 0x7fff (?B):
  Dynamic memory

  File system records

  vga.c's vga_pos

0x0505 - 0x0506 (2B):
  vga.c's vga_mask
