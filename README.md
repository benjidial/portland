# Portland OS
Portland OS is a hobby OS being developed by [Benji Dial](https://github.com/benjidial).  It is for [the i386 architecture](https://en.wikipedia.org/wiki/IA-32) and is currently in alpha.  
For documentation, see <https://benjidial.github.io/portland>.  
For bug reports, go to <https://github.com/benjidial/portland/issues>.  
To clone the source code, clone <https://github.com/benjidial/portland.git>.
